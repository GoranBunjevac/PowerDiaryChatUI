# PowerDiaryChat UI

The Angular CLI requires a minimum Node.js version of either v14.20, v16.13 or v18.10

## Description
Simple UI project written in Angular to show chat history and filter it by minute and by hour

## How to run project
- Firstly clone **PowerDiaryChatAPI** project and start the project from Visual Studio <br>
- Clone this repository, open it with Visual Studio Code (or code editor by choice) <br>
- Run `npm install` to install required dependencies <br>
- Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`
